# Parametric MIDI Sequencer — Web Prototype

## Local dev

### Backend

```bash
cd server/ParametricMusic.Api
dotnet run
```

- Swagger UI: `http://localhost:{port}/swagger`
- Health check: `GET http://localhost:{port}/health`

### Frontend

```bash
cd client
npm install
npm run dev
```

- App: `http://localhost:5173`

### Lint (frontend)

```bash
cd client
npm run lint
```
